Pig vs Cow LastZ alignments

Pig (Sus scrofa, Sscrofa9) and Cow (Bos taurus, UMD3.1) were aligned using the LastZ alignment algorithm (LastZ) in Ensembl release 64. Pig was used as the reference species. After running LastZ , the raw LastZ alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.
