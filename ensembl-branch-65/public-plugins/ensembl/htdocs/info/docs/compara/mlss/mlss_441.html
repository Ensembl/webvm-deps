Chicken vs Zebra Finch BlastZ alignments

Chicken (Gallus gallus, WASHUC2) and Zebra Finch(Taeniopygia guttata, taeGut3.2.4) were aligned using the BlastZ alignment algorithm (Schwartz S et al., Genome Res.;13(1):103-7, Kent WJ et al., Proc Natl Acad Sci U S A., 2003;100(20):11484-9) in Ensembl release 57. Chicken was used as the reference species. After running BlastZ, the raw BlastZ alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.
