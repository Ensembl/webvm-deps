Mouse vs X.tropicalis Translated Blat alignments

Mouse (Mus musculus, NCBIM37) and X.tropicalis (Xenopus tropicalis, JGI_4.2) were aligned using the Translated Blat alignment algorithm (Kent W, Genome Res., 2002;12(4):656-64) in Ensembl release 62. Mouse was used as the reference species. After running Translated Blat , the raw Translated Blat alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.
