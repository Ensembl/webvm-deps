Changes to Ensembl Mailing Lists

In 2012 we changed the way that the ensembl-dev and ensembl-announce mailing lists are managed.

The following list addresses changed with the move:

  • ensembl-dev@ebi.ac.uk became dev@ensembl.org
  • ensembl-announce@ebi.ac.uk became announce@ensembl.org

All subscribers to the old lists have been moved to the new lists. Information about the new lists, including details on how to subscribe and unsubscribe, can be found at http://lists.ensembl.org/.

As before, dev@ensembl.org is an open list which any subscriber can post to. Posting to announce@ensembl.org is restricted to Ensembl people.
