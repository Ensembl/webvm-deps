FANTOM Consortium: CAGE tags of PolyA+ RNA

Mapping of transcript tags (single 5' tags)

Description

This track shows the start (5') sequence tags of mRNA transcripts determined by the CAGE (cap-analysis gene expression) technology. The tags are composed of 21 bases on average. See the References sections below for more details on CAGE. Clicking on the feature brings up a menu with information about the tag and a link to this more detailed description.

Alignments

The tags were aligned to the unmasked genome sequence using Exonerate. The hits were filtered using the following strategy:
  • score >= 80/85, considering only highest scoring hits
  • if the tags align to more than 4 genomic locations, they were discarded as being unreliable

Credits

The ditag libraries were downloaded from the FANTOM consortium site and processed using code of the EnsEMBL project.
