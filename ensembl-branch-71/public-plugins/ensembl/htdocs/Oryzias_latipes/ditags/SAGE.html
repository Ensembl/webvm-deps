Medaka: University of Tokyo 5'-SAGE

Description

This track shows the start sequence tags of mRNA transcripts determined by 5'-end Serial Analysis of Gene Expression (5'-SAGE) sequencing. These 5'-SAGE tags are composed of 19 bases from the 5'-end of a cDNA. See the Human tag pages for more details on 5'-SAGE sequencing.

5'-SAGE libraries

  • ACAAA from medaka 0 - 7-day-old embryos and adult body tissues. FTP link

The 5'-SAGE tag features are represented as small boxes. Clicking on the feature brings up a menu with information about this tag and a link to this more detailed description.

Alignments

The 5'-SAGE tags were aligned to the unmasked genome sequence using Exonerate. The hits were filtered using the following strategy:

  • score >= 90, considering only highest scoring hits
  • if a tag aligned to more than 4 genomic locations, it was discarded as being unreliable
  • if a tag aligned to a genomic location with more than one mismatch, it was discarded as being unreliable
  • sequences with non-(ATGC) characters were removed
  • sequences with poly(ATGC) > 5 were removed

Credits

The 5'-SAGE libraries were produced at the University of Tokyo and Bioinformatics Institute. The data was mapped and analyzed using code from the EnsEMBL project.
